# Make Acheron Great Again
![Alt text](Acheron.png?raw=true "Acheron")
![License](https://img.shields.io/badge/license-GPL%20v3-blue)

## Description

A simple Honkai Star Rail relic sets comparator for Acheron.

## Features

It calculates the percentage difference of maximum damage for different relic sets, with customized teammates, eidolon, and light cones.

You can use this result as a guidance for farming.

This comparator may extend to other characters in the future.
